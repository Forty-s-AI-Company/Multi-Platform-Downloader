# PRD｜跨平台轉影音/音頻/字幕 GUI 工具（MVP）

## 1. 目標

做一個「貼網址就能下載」的桌面 GUI 工具，支援：
- 影片/音頻下載與轉檔
- 字幕下載（含自動字幕）
- 清晰度/音質選擇
- 播放清單下載
- 區間下載（開始/結束時間）

核心原則：**先把 `yt-dlp + ffmpeg` 的能力做成穩定、好用的 UI**。

## 2. 主要使用者故事

1. 我貼一個 YouTube 影片網址 → 看到可選清晰度/音質 → 下載成 mp4
2. 我貼一個 Shorts/TikTok/IG Reels 連結 → 一鍵「最佳」下載
3. 我只想要音訊 → 下載成 mp3（可選高音質）
4. 我想拿字幕 → 下載字幕（手動/自動）並轉成 srt
5. 我貼播放清單 → 自動分資料夾下載整串
6. 我只想下載某一段（例如 00:01:10-00:03:40）→ 成功產出切好的 mp4

## 3. 平台範圍（以能力分層）

### A. URL 直接支援（通常可用）
- YouTube（含 shorts、playlist）
- TikTok（部分內容可能需 cookies）
- Douyin（支援度依版本與地區）
- Instagram Reels / 貼文影片（多數情況需 cookies/登入）
- 其他 `yt-dlp` 已支援站點（後續擴充）

### B. 需要 cookies / 可能失效（UI 要提示）
- IG / FB / 部分直播
- Skool（桌面參考做法：Playwright 抓 m3u8 + yt-dlp 下載）

> `docs/PLATFORM_SUPPORT.md` 會把各平台的注意事項寫清楚。

## 4. 功能需求（MVP 必做）

### 4.1 下載工作（Job）
- 新增 Job：輸入 URL、選輸出路徑、選項（畫質/音質/字幕/播放清單/區間）
- Job 狀態：排隊/下載中/轉檔中/完成/失敗
- 顯示進度：速度、ETA、已下載大小
- 失敗要顯示可讀訊息（不要只丟一坨 stderr）

### 4.2 清晰度/音質選擇
- **簡單模式：**
  - 影片：最佳 / 1080p / 720p / 480p
  - 音訊：最佳 / 一般
- **進階模式：**
  - 顯示 `yt-dlp -F` 的格式清單
  - 允許輸入格式碼（例如 `399+140`）

### 4.3 字幕
- 可選：寫入字幕、寫入自動字幕
- 語言：預設 `zh-Hant,en`（可自訂）
- 輸出：可選轉成 `srt`

### 4.4 播放清單
- 下載整個 playlist（預設開啟 `--yes-playlist`）
- 輸出路徑按 playlist title 分層

### 4.5 區間下載
- 輸入 `開始時間` / `結束時間`（`HH:MM:SS`）
- 對應 `yt-dlp --download-sections "*START-END"`

### 4.6 Cookies / 登入
- 匯入 `cookies.txt`
- 可選從瀏覽器讀取（至少 Chrome），並清楚提示風險

## 5. 非功能需求

- 跨平台：Windows/macOS/Linux（MVP 先以 Windows 開發驗證）
- 可觀測性：log 需可關閉敏感資訊
- 下載不中斷：GUI 關閉後（可選）仍可繼續（後續）
